  This example is based on internal test board, it shows how to use SDIO to read or write 
to eMMC. After initializing the card successfully, print out the detailed informationof 
the card by USART. Then write a block of data to the card and read. If any error occurs, 
print the error message. After that,  it is the multiple blocks operation test. If any error occurs, 
print the error message. If no error occurs, print the ending message.

  Uncomment the macro DATA_PRINT to print out the data and display them through HyperTerminal.
